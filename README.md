# Portal PyEmulator

> _Special thanks to **Marijn Kneppers** (@mandar1jn) for publishing the research article:
> [Reverse engineering Skylanders' Toys-to-life mechanics](https://marijnkneppers.dev/posts/reverse-engineering-skylanders-toys-to-life-mechanics/)
> which served as the foundation for the `portal_io/` folder, and making this whole project possible._

Portal PyEmulator is a Python-based project for emulating the Traptanium Portal of Power
or interfacing any Portal of Power via man-in-the-middle.

It connects between RPCS3 and the Portal of Power, allowing it to inspect, modify, and inject data into the USB stream.
